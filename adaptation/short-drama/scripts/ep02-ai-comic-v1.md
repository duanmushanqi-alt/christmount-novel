# 第2集：守灵

## 基本信息

- 时长：约 180 秒
- 形式：AI 漫剧画面脚本
- 核心看点：黄简用鬼神话术争取守灵，在老者十年影响中抓住换尸机会
- 本集功能：承接 EP1 结尾，完成守灵谈判、守灵情绪落点、换尸伪装和草席包裹
- 结尾钩子：第二夜狱卒回来搬尸，黄简已经裹入草席，只能屏息等门开

## EP1 承接点

- EP1 结尾停在年轻狱卒俯身要拖老者尸体，黄简半跪护住老者，冷声说：“想死，你就动他。”
- 黄简已经被年轻狱卒踹打，身体极虚，但眼神和判断仍稳。
- 年轻狱卒粗躁、怕晦气；老狱卒冷硬、图省事，但也懂牢里的忌讳。
- 老者临终只留下“青松庙”“留过些东西”“好好活下去”，本集不扩展藏宝信息。
- EP2 不使用旁白，不补完整前因，只通过动作、短对白和狱卒反应推进。

## 视听原则

- 延续 EP1 的死牢质感：滴水、火把、铁链、脚步声。
- 鬼神话术保持真假暧昧，画面只呈现风、灯、影和狱卒心理反应，不拍成真灵异。
- 黄简的悲痛只短暂失控，随后立刻转入求生判断。
- 换尸细节要清楚，但不猎奇；重点是手抖、剪发、换衣、草席包裹和屏息待搬。

## 画面脚本

### 画面1

画面：承接 EP1 结尾。年轻狱卒的手悬在老者脚踝上方，老者身下露出破草席卷边。黄简半跪在老者身侧，手压着老者衣摆，抬眼看他。
景别：三人中近景
建议时长：5 秒
台词：
黄简：想死，你就动他。
字幕：无
音效：低频鼓点余音、滴水声突然清晰
动效：镜头从年轻狱卒悬住的手慢慢推到黄简的眼睛
备注：不重播太长，只把 EP1 的钩子接稳。

### 画面2

画面：年轻狱卒僵了一瞬，随即恼羞成怒，伸手揪住黄简的衣领，把他往墙上一搡。
景别：双人近景
建议时长：6 秒
台词：
年轻狱卒：你敢吓唬我！
字幕：无
音效：衣领被抓紧、后背撞墙闷响、铁链轻震
动效：黄简撞上墙，火把光在两人脸上一晃
备注：年轻狱卒不能立刻被吓住，先用怒气压恐惧。

### 画面3

画面：黄简被搡得肩背发抖，却没有挣扎。他的视线从年轻狱卒的脸移到地上的老者。
景别：黄简视线特写
建议时长：7 秒
台词：
黄简：他刚咽气。
字幕：无
音效：滴水声、年轻狱卒急促呼吸、老者衣摆轻轻垂落
动效：镜头跟随黄简视线，落到老者半合的眼和被拖乱的衣摆
备注：从“刚死”这个事实起话，贴近正文，不凭空造规则。

### 画面4

画面：年轻狱卒正要骂，老狱卒举着火把站在后方，火光照到老者苍白的脸和牢墙上的湿苔。
景别：老者面部近景转老狱卒近景
建议时长：7 秒
台词：
黄简：魂还没走。
字幕：【重点】魂还没走。
音效：火把噼啪、远处风声钻入牢窗
动效：火光忽明忽暗，老狱卒的眉头收紧，年轻狱卒的骂声停在喉咙里
备注：用正文里的“魂还没走”替代额外民俗设定。

### 画面5

画面：年轻狱卒强撑着冷笑，抬脚要再踹黄简。脚刚抬起，小窗外一阵风灌入，火把火苗猛地一低。
景别：中近景
建议时长：7 秒
台词：
年轻狱卒：活人我都不怕，还怕死人？
字幕：无
音效：衣摆被风掀动、火把噼啪一声
动效：火光一暗，年轻狱卒的影子在墙上拉长
备注：风和火光都可解释，作用是放大狱卒心虚。

### 画面6

画面：黄简被阴影压着，声音更低。
景别：黄简特写
建议时长：8 秒
台词：
黄简：你们都知道，他以前不简单。这时候动他，他会缠上动手的人。
字幕：无
音效：滴水声变慢，铁链轻轻晃动
动效：镜头略微下移，扫过老者被拖过半寸的痕迹，再停在年轻狱卒悬住的脚
备注：第一次轻点老者异样，不解释，只让狱卒自己想起过去的忌惮。

### 画面7

画面：年轻狱卒喉结动了一下，嘴上仍硬。他把黄简往地上一推，黄简摔在破草席边，年轻狱卒的眼神却避开老者。
景别：双人中景
建议时长：7 秒
台词：
年轻狱卒：少吓唬人。
字幕：无
音效：身体摔地、稻草摩擦
动效：黄简摔到稻草上，手仍下意识护向老者衣摆
备注：年轻狱卒外强中干，嘴上压人，身体已经不愿再靠近老者。

### 画面8

画面：老狱卒没有立刻回答。他举着火把靠近老者，火光照到老者半合的眼、灰白的脸、塌边的旧灰帽、身下破草席，还有墙角几道谁也看不懂的旧刻痕。
景别：老者面部近景
建议时长：7 秒
台词：
老狱卒：你想干什么？
字幕：无
音效：火把噼啪、老狱卒压低的呼吸
动效：火光扫过老者眼缝、灰帽和墙角刻痕，墙上影子像轻轻动了一下
备注：老狱卒不接“怕不怕”，只问目的，给黄简开出方案的口子。

### 画面9

画面：黄简撑着手坐起，掌心压在老者身下露出的破草席边。他看着老狱卒，不看年轻狱卒。
景别：黄简近景
建议时长：6 秒
台词：
黄简：让我守一天，算给他送行。
字幕：【重点】让我守一天。
音效：稻草被抓紧、滴水声
动效：镜头从黄简压住草席的手慢慢推到他的眼睛
备注：先给请求本身，不让长台词压在单一近景上。

### 画面10

画面：年轻狱卒下意识看向老者。火光扫过老者半合的眼、旧灰帽、破草席卷边，又扫进牢门外黑沉沉的走廊。
景别：狱卒反应中景
建议时长：7 秒
台词：
黄简：他安心走，怨气自然就散了。还是你们想自己守？
字幕：无
音效：火把噼啪、年轻狱卒吞咽声
动效：年轻狱卒脚尖往后挪半步，老狱卒没有看他，只盯着老者
备注：后半句压在狱卒反应上，让威胁成立。

### 画面11

画面：年轻狱卒刚要反驳，老狱卒抬手拦住他。
景别：老狱卒中近景
建议时长：6 秒
台词：
老狱卒：算了。走，明晚再来。
字幕：无
音效：钥匙串轻轻一响
动效：老狱卒的手挡在年轻狱卒胸前，年轻狱卒的话被压回去
备注：老狱卒先结束冲突，不在牢内解释太多。

### 画面12

画面：年轻狱卒不甘心，临出门前回头指着黄简，手指隔着铁栏停了一下，又硬生生收回去。
景别：牢门内外中景
建议时长：6 秒
台词：
年轻狱卒：臭小子，你少耍花样。
字幕：无
音效：脚步后退、铁栏轻响
动效：年轻狱卒的手指在铁栏外僵住一拍，随即被老狱卒拉走
备注：删去年轻狱卒毁坏器物的动作，避免后续道具逻辑问题。

### 画面13

画面：两名狱卒退出牢门。铁门合上，锁扣压死。年轻狱卒隔着铁栏回头看黄简和老者，声音压得很低。
景别：牢门内外中景
建议时长：7 秒
台词：
年轻狱卒：咱们就这么走了？
字幕：无
音效：落锁声、钥匙轻响、火把噼啪
动效：火把光随着狱卒退出，牢内迅速暗下去，只剩小窗冷光
备注：出门、锁门、再低声问，动作和台词顺序更合理。

### 画面14

画面：两名狱卒沿走廊走远几步，老狱卒的火把光隔着铁栏晃过墙角旧刻痕。
景别：走廊中景
建议时长：7 秒
台词：
老狱卒：你又不是不知道，那老头以前就神神叨叨的。
字幕：无
音效：脚步声、火把噼啪
动效：镜头贴着铁栏缝，看老狱卒的背影和牢内黄简的影子错开
备注：退避理由放在门外说，不做设定说明。

### 画面15

画面：两名狱卒继续走远，火把光越来越弱，死牢重新安静。
景别：走廊中景转牢内中景
建议时长：7 秒
台词：
老狱卒：这牢也邪性。多一事不如少一事。他想守就守，明晚再处理。
字幕：无
音效：脚步声渐远、滴水声重新清晰
动效：火把光消失后，镜头慢慢转回牢内，黄简仍保持撑坐姿势
备注：用离开时的低声交谈解释他们退避的真实原因：忌讳、偷懒、怕担责。

### 画面16

画面：黄简保持半跪姿势不动，直到脚步声完全消失。他的肩忽然塌下去，整个人伏到老者身旁。
景别：中近景
建议时长：6 秒
台词：无
字幕：无
音效：远处脚步断掉、黄简压抑喘息
动效：黄简撑住很久，确认安全后才失力
备注：让观众看见他刚才一直硬撑。

### 画面17

画面：黄简的手在稻草里摸到一块磨薄的石片。闪回：多年以前，老者从草席下摸出同样的石片，递到年轻些的黄简面前。黄简背过身，脸上带着被打扰的不耐烦。
景别：手部特写转闪回近景
建议时长：7 秒
台词：
老者旧声：这块我磨了很久，收着吧。也许哪天就用上了。
字幕：无
音效：石片轻响、旧日滴水声叠入
动效：石片从老者掌心递到黄简手边，再叠化回现在
备注：先完成石片来源交代，不急着展开两人关系。

### 画面18

画面：回到现在。黄简低头看着掌心里的磨薄石片，又看向老者安静的脸。小窗冷光落在两人之间，牢房重新只剩滴水声。
景别：石片与老者近景
建议时长：6 秒
台词：无
字幕：无
音效：当前滴水声、压低的呼吸
动效：镜头从石片边缘慢慢移到老者脸侧
备注：石片交接后回到当下，给黄简做决定前的停顿。

### 画面19

画面：黄简的手指一点点收紧，石片边缘压进掌心。他额头抵在稻草上，呼吸从乱到稳。
景别：黄简手部与侧脸特写
建议时长：6 秒
台词：无
字幕：无
音效：石片轻轻摩擦掌心、压低的呼吸
动效：手指收紧，黄简眼神从闪避变得清醒
备注：决定前的内在转折，避免闪回堆在决定之前过长。

### 画面20

画面：黄简抬头看向老者，眼神定住。镜头再拉远，两人的影子在墙上几乎叠在一起。
景别：黄简脸部特写转墙影中景
建议时长：7 秒
台词：
黄简：师傅，再帮我这一次。
字幕：【重点】师傅，再帮我这一次。
音效：压低的呼吸、滴水声、极轻的铁链声
动效：先停在黄简眼睛，再拉到墙上重叠的影子
备注：先下定决心，再进入割发和闪回。

### 画面21

画面：分四个短镜头。1. 磨薄石片贴近黄简自己的发根，一缕黑发断落。2. 闪回里，老者靠墙絮絮低念。3. 年轻些的黄简先别过脸，忍到最后猛地转回头。4. 老者停下低念，第一次认真看他。
景别：割发特写 / 老者近景 / 黄简侧脸近景 / 双人中近景
建议时长：11 秒
台词：
老者旧声：五行看生克，八卦看方位。时位错了，活路也会变死路。
黄简旧声：你到底天天念叨什么？
老者旧声：你听懂了？
黄简旧声：没听懂。你天天说，天门到底是什么？
老者旧声：天门，是阴阳转关的地方。
字幕：无
音效：石片割发、旧日滴水声叠入
动效：黑发落下，叠化为旧日老者抬眼
备注：用一句通俗但有古法质感的五行八卦内容建立老者的异质感；台词按短镜头分配，不让一张画面承载全部对白。

### 画面22

画面：切为四个短镜头。1. 闪回里，黄简坐直，第一次主动看向老者。2. 老者抬眼，神色仍淡，停了一息，像是在判断他是不是真想学。3. 黄简往前挪近半步，眼神不再躲。4. 回到现在，黑发落进稻草里，灰白发从老者鬓边断落；旧灰帽和破草席留在黄简手边。
景别：闪回双人中近景 / 老者近景停顿 / 黄简近景 / 道具近景
建议时长：11 秒
台词：
黄简旧声：我想学。
老者旧声：学这个，要先学守心。术无善恶，善恶在人心。
黄简旧声：那你教我。
老者旧声：好，从今日起，你就算我的徒弟。
字幕：无
音效：滴水声持续，远处白日杂音短暂浮起又退下，石片割发、草席摩擦
动效：同一小窗光色变化，道具位置在切帧间逐次改变
备注：补足“师傅”称呼来源，但不做完整拜师仪式；老者说“好”之前留一个判断黄简的停顿，避免答应得太轻。

### 画面23

画面：天色转暗前，黄简把自己的囚衣替老者穿好，把剪下的几缕黑发压在老者额前阴影里。随后他扶老者坐到墙角，头垂着，脸被阴影遮住，只剩几缕黑发贴在额前。黄简退开一步，从门口方向看去，墙角只像一个低头昏睡的囚犯。
景别：肩背近景 / 额前头发特写 / 墙角中景
建议时长：9 秒
台词：无
字幕：无
音效：衣料摩擦、压住的呼吸声、远处模糊的巡夜声
动效：先套囚衣，再把几缕黑发压进阴影，最后扶老者坐入墙角阴影
备注：老者此后尽量只作为阴影里的人形出现，不拍清晰外貌；几缕黑发只作为伪装轮廓提示，不做假发式特写。

### 画面24

画面：牢内大全景。墙角坐着穿黄简囚衣的老者，整个人陷在阴影里。黄简站在小窗前，已经换上老者旧衣，抬手戴上旧灰帽，灰白发垂到脸侧。破草席半摊在地上，离他只有一步。
景别：牢内大全景
建议时长：8 秒
台词：无
字幕：无
音效：滴水声、衣料最后一点摩擦声停住、压低的心跳声开始出现
动效：大全景固定不动，只让黄简把帽檐压低；小窗光把他变成背影
备注：这一帧是身份互换的视觉确认：角落是“像黄简的人”，窗前是即将变成“老者尸身”的黄简。保持背光和阴影，避免看清脸。

### 画面25

画面：分三个短镜头。1. 俯拍，黄简躺到半摊开的草席上，身体蜷紧。2. 草席边缘从画面一侧卷过来，遮住他的肩背和旧衣。3. 草席口只露出一点旧灰帽边和灰白发，边缘随着一口紧张的呼吸轻轻起伏，又被他强行压住。
景别：草席俯拍中景 / 草席口近景 / 边缘极近景
建议时长：8 秒
台词：无
字幕：无
音效：草席摩擦、紧张呼吸、心跳声逐渐压低
动效：三段式推进：躺下、遮住肩背、草席口静止
备注：不要生成一镜到底的复杂卷身动作；用三个短镜头表达黄简自己卷入草席。不露眼、不露脸，把紧张点放在呼吸和心跳。

### 画面26

画面：小窗外天色彻底黑下。走廊尽头先响起脚步声，随后钥匙相撞。火把光从门缝扫进来，先扫过墙角垂头坐着的“黄简”，再停在草席边缘。草席原本极轻的起伏消失。钥匙插进锁孔，铁锁将开未开，切黑。
景别：门缝光线近景 / 牢内中景 / 锁孔特写 / 草席边缘特写
建议时长：7 秒
台词：无
字幕：无
音效：巡夜梆声远去、脚步声由远及近、钥匙碰铁、火把噼啪、锁舌轻响
动效：火把光线先扫墙角，再扫草席；草席完全静止；锁舌一响立刻切黑
备注：集尾钩子停在开门前一瞬。下一集可从门开接入，狱卒一句“还挺讲究，自己裹上了”带过草席。

## 本集声音设计

### 角色音色

- 黄简：低、哑、短句；吓狱卒时压低声音，让对方自己害怕。
- 年轻狱卒：前半段急躁逞强，后半段多停顿和吞咽声。
- 老狱卒：冷、稳、现实；他信一点忌讳，但更在意省事和不担责。

### 声音层次

1. 环境音：滴水、火把、铁锁、走廊风声。
2. 人声：前半段对白密，守灵后几乎无对白。
3. 音效：铁栏轻响、石片轻响、石片割发、衣带摩擦、草席摩擦、脚步回归。
4. BGM：前半段悬疑低频，守灵段降到极弱，结尾脚步声重新压上来。

## 本集字幕建议

- 字幕保持短句，不加解释性字幕。
- “魂还没走。”可做前半段重点字幕。
- “让我守一天。”单独卡点显示。
- “师傅，再帮我这一次。”要压低处理，不做热血处理。
- 结尾不加解释性字幕，用小窗光线变化、巡夜梆声和脚步声完成第二夜转场。

## 本集自检

- 是否承接 EP1：是，直接从“想死，你就动他”接入守灵谈判。
- 是否有本集推进：有，黄简争取到一天，并在老者十年影响的回声中完成换尸伪装和草席包裹。
- 是否保持克制：是，黄简不喊狠话，关键情绪只三句低声台词。
- 是否保留智谋感：是，鬼神话术来自“刚咽气”“魂未走”“怨气未散”和牢中忌讳。
- 是否避免真灵异：是，所有异样都可由风、灯、影和狱卒心虚解释。
- 是否有结尾钩子：有，第二夜狱卒回来搬尸，黄简已经裹入草席，只能屏息等门开。
- 是否适合 AI 漫剧生产：适合，拆成 26 个画面，闪回、蒙太奇和结果帧都避免复杂连续动作。

# EP1 人物形象设计基准

> 用途：给设计师、文生图 AI、角色一致性参考使用。  
> 范围：第 1 集《死人堆里的人》主要出场人物。  
> 原则：只服务漫剧制作，不修改小说正文设定。

## 统一画风

- 架空古代，写实质感，偏电影分镜 / 精品漫剧。
- 色调：低饱和、冷灰、惨白月光、昏黄油灯。
- 氛围：阴冷、潮湿、压抑、克制。
- 人物要有“脏、冷、瘦、旧、压抑”的真实感。
- 不要玄幻光效，不要仙侠飘逸，不要爽文邪魅表情。

## 黄简｜死囚状态

### 角色定位

从死人堆里爬出来的主角。身体濒临崩溃，但判断没有崩。  
核心气质是虚弱、克制、隐忍，不是爽剧式强势外放。

### 外形

- 二十多岁到三十岁出头。
- 极瘦，长期牢狱导致肩背单薄、脸颊凹陷。
- 脸色灰白，嘴唇干裂，眼窝深。
- 头发凌乱、半湿半脏。
- 脸上有血泥、灰尘、冷汗。
- 小臂有旧伤和新擦伤。
- 手指瘦，指甲里有泥。

### 服饰

- 破旧囚衣或不合身旧衣。
- 粗布材质，发灰，边缘磨损。
- 衣服有泥、血迹、草屑。
- 不能干净、合身、精致。

### 表情范围

- 虚弱喘息。
- 压着痛。
- 强忍悲意。
- 冷冷抬眼。
- 不哭喊，不狂怒，不邪笑。

### 中文提示词

```text
古代架空死囚青年，极瘦，脸色灰白，眼窝深陷，嘴唇干裂，头发凌乱，脸上有血泥和冷汗，破旧粗布囚衣，手腕有旧伤和新擦伤，神情虚弱但克制，眼神冷静隐忍，惨白月光，低饱和写实电影感
```

### 负面提示词

```text
不要俊美干净，不要邪魅狂狷，不要玄幻发光，不要现代服装，不要肌肉壮硕，不要夸张愤怒表情，不要霸气外露
```

## 老者｜临终托付者

### 角色定位

牢中将死之人。不是仙风道骨的高人，而是被困多年、临终前把自己用不上的东西托给黄简的人。

### 外形

- 七十岁左右。
- 极瘦，皮肤松弛，颧骨突出。
- 白发稀疏凌乱。
- 胡须花白，但不飘逸。
- 手枯瘦，指节明显。
- 抓住黄简手腕时有最后一点力气。
- 眼神浑浊、疲惫，但有一点温和和托付感。

### 服饰

- 破旧灰白囚衣。
- 袖口磨烂，布料发硬发脏。
- 头上戴一顶灰白破布帽，帽檐塌软，帽身有补丁和污渍。
- 帽子要像牢中老人长期戴旧的保暖破帽，不是道士冠、僧帽或高人头饰。
- 躺在潮湿死牢的稻草上。

### 表情范围

- 气若游丝。
- 艰难开口。
- 轻轻摇头。
- 眼神慢慢散开。
- 不神秘，不微笑，不像仙人。

### 中文提示词

```text
古代牢中垂死老者，七十岁左右，枯瘦，白发稀疏凌乱，胡须花白，脸色灰败，颧骨突出，戴灰白破布帽，帽檐塌软，帽身有补丁和污渍，穿破旧灰白囚衣，躺在潮湿死牢稻草上，眼神浑浊但温和，临终托付，昏暗油灯与小窗月光，写实压抑质感
```

### 负面提示词

```text
不要仙风道骨，不要道士法袍，不要道士冠，不要僧帽，不要精神矍铄，不要发光，不要神秘高人姿态，不要干净整洁
```

## 年轻狱卒

### 角色定位

粗暴、急躁、麻木的底层差役。不是武侠反派，也不是高级打手。

### 外形

- 二十多岁到三十岁。
- 身形结实但不精壮。
- 脸有粗粝感。
- 眉眼不耐烦。
- 嘴角常带嫌恶。
- 动作粗鲁，踢、拽、拍手都带着麻木和不耐烦。

### 服饰

- 北魏死牢狱卒服。
- 深灰或暗褐短袍。
- 腰间钥匙串。
- 衣服脏旧，有油渍、灰尘。
- 可搭配饭桶、饭勺。

### 表情范围

- 不耐烦。
- 嫌晦气。
- 被冒犯后的恼怒。
- 粗暴压人。
- 不要奸笑，不要精致反派脸。

### 中文提示词

```text
古代死牢年轻狱卒，二十多岁，粗粝结实，表情不耐烦，眉眼凶躁，穿深灰旧狱卒短袍，腰间钥匙串，手边饭桶和饭勺，动作粗暴，昏黄油灯，潮湿牢房，写实低饱和
```

### 负面提示词

```text
不要武侠高手造型，不要盔甲将军感，不要夸张反派笑，不要现代警服，不要华丽服饰，不要精致英俊
```

## 老狱卒

### 角色定位

冷硬、麻木、熟悉牢房规矩的人。比年轻狱卒更沉稳，也更冷。

### 外形

- 四十多岁到五十岁。
- 脸瘦硬，皱纹深。
- 眼神冷淡麻木。
- 不咋呼，不多话。
- 拿油灯检查死亡时动作熟练。

### 服饰

- 深色旧狱卒服，比年轻狱卒更旧、更沉。
- 腰间钥匙串。
- 手持油灯。
- 整体像长期待在牢里的老差役。

### 表情范围

- 冷淡。
- 皱眉。
- 嫌麻烦。
- 例行公事。
- 不慈祥，不像正派长者。

### 中文提示词

```text
古代死牢老狱卒，四五十岁，瘦硬脸，皱纹深，眼神冷淡麻木，穿深褐旧狱卒服，腰间钥匙串，手持昏黄油灯，动作熟练，潮湿阴冷死牢，写实电影质感
```

### 负面提示词

```text
不要慈祥老人，不要官员华服，不要武将盔甲，不要夸张凶恶，不要现代服饰，不要仙侠感
```

## 角色一致性重点

- 黄简：瘦、灰白、脏、冷静。
- 老者：枯瘦、临终、温和但不神秘。
- 年轻狱卒：粗躁、嫌恶、动作大。
- 老狱卒：冷硬、熟练、嫌麻烦。

## 制作建议

先分别生成四名人物的正面半身定妆图，再进入 EP1 分镜图生成。  
定妆图优先统一以下维度：

- 面部年龄。
- 体态胖瘦。
- 服饰颜色和材质。
- 头发凌乱程度。
- 污渍、血泥、伤痕位置。
- 表情基准。

EP1 分镜生成时，黄简必须保持“虚弱但眼神稳”的一致性；老者必须保持“临终而非神秘高人”的一致性；两名狱卒必须保持“底层差役”的现实质感。

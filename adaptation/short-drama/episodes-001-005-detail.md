# EP1-5 精细分集表

## 版本信息

- 版本：v1
- 对应目录：`episodes-outline-v1.md`
- 集数范围：EP1-5
- 单集时长：主方案 3 分钟；可扩展至 5 分钟
- 功能定位：样片段 / 开篇五集

## 开篇五集总目标

前五集必须完成三件事：

1. 用乱葬岗冷开场抓住观众，让观众立刻知道“这个人是从死人堆里爬出来的”。
2. 保留原作最有差异化的智谋点：守灵、换尸、鬼神话术，而不是写成普通逃亡戏。
3. 在第五集结束时，让黄简完成第一阶段身份转换：从死囚黄简变成陈汉商人黄繁。

## 开篇五集硬约束 v2

1. 不使用大段旁白解释前情；必要信息通过画面、动作、短对白或回切场景交代。
2. 可以交代黄简是“通敌犯”“死囚”“被关十年”，但不提前讲透完整阴谋链。
3. 老者和黄简的关系用动作建立，不靠长篇师徒回忆。
4. 黄简身体可以濒临崩溃，判断不能崩；情绪可以被刺痛，行动不能失控。
5. 每集结尾必须有明确钩子，钩子来自处境、危险、未解信息或未完成行动，不靠廉价意外。
6. 前五集必须完成“死囚黄简 → 陈汉商人黄繁”的身份转换闭环。
7. 必须保留守灵换尸的智谋感，不能写成普通逃亡或纯运气逃生。
8. 鬼神元素保持真假暧昧但逻辑可解释，不能写成真玄幻。
9. 黄简可以做灰色求生行为，但不能让观众觉得他失去底线。
10. 前五集不亮陈伯庸真名，只露“权力阴影”和“刺史亲戚”一类压迫信息。

## EP1 死人堆里的人

### 本集功能

用最强视觉钩子打开故事：黄简从乱葬岗爬出。随后回切死牢，让观众知道这不是奇观，而是一场刚刚发生的换命局。

### 3 分钟节奏

| 时间段 | 内容 |
|---|---|
| 0:00-0:35 | 冷开场：乱葬岗，草席轻动，黄简从死人堆里爬出 |
| 0:35-0:50 | 黄简在惨白月光下喘息，抬小臂遮眼；从手腕旧伤叠化回死牢 |
| 0:50-1:32 | 死牢中，老者临终，只托出“青松庙”和“好好活下去” |
| 1:32-1:42 | 黄简安置老者，退到墙角，压住悲痛 |
| 1:42-2:25 | 狱卒送饭，发现异常，进牢确认老者已死 |
| 2:25-3:00 | 狱卒要拖走老者，黄简制止，被打后再次制止，结尾冷声威胁 |

### 核心场景

1. 乱葬岗冷开场
2. 死牢老者临终
3. 狱卒进牢

### 必须传达的信息

- 黄简是从尸体堆里逃出来的人。
- 他被关在死牢很久，身体极虚弱。
- 老者是他在狱中的重要之人。
- 老者只留下“青松庙”和“留过些东西”的种子，不交代具体位置和开启方式。
- 黄简保护老者尸身，第一次显出压住痛苦后的冷硬。

### 情绪点

- 第一口自由空气。
- 老者死去时黄简的克制悲痛。
- 狱卒粗暴对尸体，黄简忍住没有立刻爆发。

### 结尾钩子

狱卒的手碰到老者脚踝，黄简冷声说：“想死，你就动他。”观众想知道：他一个虚弱死囚，凭什么让狱卒害怕？

### 注意事项

- 乱葬岗冷开场不要解释太多。
- 老者临终不要说成遗嘱清单，重点放在“你要活着”。
- 黄简第一集不需要说狠话。

## EP2 守灵

### 本集功能

完整呈现黄简如何用鬼神话术争取守灵，并开始换尸。核心是智谋和压迫，不是动作戏。

### 3 分钟节奏

| 时间段 | 内容 |
|---|---|
| 0:00-0:25 | 承接 EP1，狱卒不耐烦，要直接拖尸 |
| 0:25-1:10 | 黄简以“怨气”“守灵”“晦气”吓住年轻狱卒 |
| 1:10-1:35 | 老狱卒权衡，决定让黄简守一夜，自己省事 |
| 1:35-2:20 | 狱卒离开，黄简为老者整理遗容，情绪短暂失控 |
| 2:20-2:50 | 黄简意识到换尸机会，开始剪发、换衣、处理痕迹 |
| 2:50-3:00 | 结尾钩子：狱卒脚步声回来，黄简还没完全处理完 |

### 核心场景

1. 黄简吓狱卒
2. 黄简守灵
3. 换尸准备

### 必须传达的信息

- 黄简懂一些鬼神话术，不是真有神通。
- 狱卒恐惧和偷懒共同成就了这个机会。
- 黄简换尸不是临时奇迹，而是抓住微小缝隙的判断。

### 情绪点

- 黄简对老者说“对不起”，但他必须活。
- 他悲痛是真的，利用尸体也是真的。

### 结尾钩子

狱卒回来的脚步声靠近，黄简仍在处理换尸痕迹。

### 注意事项

- 守灵不是煽情戏，而是悲痛和求生叠在一起。
- 换尸细节可以视觉化，但不要拍成过度猎奇。
- 黄简行动虚弱、手抖，但判断清楚。

## EP3 鬼剃头

### 本集功能

承接换尸结果，完成逃狱暴露和追捕压力。让观众知道黄简暂时逃出死牢，但危险没有结束。

### 3 分钟节奏

| 时间段 | 内容 |
|---|---|
| 0:00-0:30 | 狱卒回来，抬走“老者”，黄简以尸体身份离开死牢 |
| 0:30-1:05 | 乱葬岗补足：黄简脱出草席，接回 EP1 冷开场之后的行动 |
| 1:05-1:45 | 第二日，狱卒回牢处理“黄简”，发现头发和尸体异常 |
| 1:45-2:20 | 新狱监震怒，压下越狱消息，下令暗中追杀 |
| 2:20-2:50 | 黄简在荒野中躲避搜捕，用简单障眼法脱身 |
| 2:50-3:00 | 结尾钩子：黄简远远看见陈汉国方向的关卡/城门 |

### 核心场景

1. 尸体被抬出死牢
2. 狱卒发现“鬼剃头”
3. 新狱监下令追杀
4. 黄简荒野躲避

### 必须传达的信息

- 黄简逃出去了，但监狱方面不打算公开追捕。
- 新狱监和陈伯庸有隐约关系，只露“刺史亲戚”。
- 黄简必须离开北魏边境，先弄到身份。

### 情绪点

- 黄简自由后的第一反应不是狂喜，而是活下去。
- 新狱监的轻蔑体现权力压迫：一个囚犯死不死都可以被抹平。

### 结尾钩子

黄简看到陈汉国方向。他要怎么以逃犯身份过关？

### 注意事项

- “鬼剃头”要作为狱卒恐惧的误读，不是真灵异。
- 新狱监不要讲太多背景，一句“刺史大人的亲戚”足够。

## EP4 逃入陈汉

### 本集功能

从荒野逃亡转入身份问题：黄简明白自己不能只靠逃，必须有一个合法身份。

### 3 分钟节奏

| 时间段 | 内容 |
|---|---|
| 0:00-0:35 | 黄简观察陈汉国边境城门盘查规律 |
| 0:35-1:10 | 他偷/借农户衣物和农具，换成农人模样 |
| 1:10-1:45 | 黄简混入宁州城，险些被盘问，用卑微话术过关 |
| 1:45-2:20 | 入住吉祥客栈，观察陈老板和客栈环境 |
| 2:20-2:50 | 黄简试探办身份，陈老板警惕拒绝 |
| 2:50-3:00 | 结尾钩子：黄简发现陈老板请来的道士进了少东家房间 |

### 核心场景

1. 城门观察
2. 换装混入
3. 吉祥客栈试探

### 必须传达的信息

- 黄简会观察，不会盲闯。
- 他能伪装成卑微农人。
- 陈汉国边境可以买身份，但不是随便能买。
- 陈老板有门路，但不信任他。

### 情绪点

- 黄简身无分文、身体虚弱，但他仍能看懂人和规矩。
- 他意识到：自由只是第一步，身份才是活下去的壳。

### 结尾钩子

陈老板少东家病重，道士无效。黄简看见了另一个机会。

### 注意事项

- 偷衣服要处理底线问题：后面 EP5 或本集中埋下“会还”的动作/念头。
- 陈老板不能太好骗。
- 黄简的风水能力不要显得像超能力。

## EP5 黄繁

### 本集功能

完成开篇五集的小闭环：黄简以知识和判断换得身份，从逃犯变成黄繁。

### 3 分钟节奏

| 时间段 | 内容 |
|---|---|
| 0:00-0:35 | 黄简观察少东家房间，指出道士忽略的问题 |
| 0:35-1:20 | 陈老板试探黄简，黄简用看相/风水推理说中关键 |
| 1:20-1:55 | 黄简调整房间布置，留下验证条件 |
| 1:55-2:20 | 时间压缩：少东家好转，陈老板态度改变 |
| 2:20-2:45 | 陈老板帮黄简办下身份：黄繁，陈汉国宁州城人士 |
| 2:45-3:00 | 结尾钩子：黄繁踏上回北魏的路，低声念出新名字 |

### 核心场景

1. 少东家房间
2. 陈老板试探
3. 身份文书到手

### 必须传达的信息

- 黄简的能力来自观察、经验和老者所教，不是神迹。
- 陈老板帮他，是因为黄简证明了价值。
- 黄简获得合法身份“黄繁”。
- 他下一步要回北魏。

### 情绪点

- 黄简第一次用学到的东西换来生路。
- 他获得身份时没有喜形于色，因为真正的路才刚开始。

### 结尾钩子

黄繁站在回北魏的路上。观众知道，他要回到那个让他家破人亡的地方。

### 注意事项

- 风水救人要写得像古代经验判断，不要像法术治病。
- 黄繁身份生成不能太轻松，要让陈老板有利益考量和风险权衡。
- 结尾不要喊复仇口号，只让他使用新名字即可。

## 3 分钟版与 5 分钟扩展位

| 集数 | 3 分钟必需内容 | 5 分钟可扩展内容 |
|---|---|---|
| EP1 | 乱葬岗冷开场、老者临终、狱卒动尸 | 增加老者与黄简的短暂日常关系碎片 |
| EP2 | 鬼神话术、守灵、换尸准备 | 增加换尸操作细节和黄简对老者的道别 |
| EP3 | 抬尸、鬼剃头、追杀、逃向陈汉 | 增加荒野躲避和狱卒追踪压力 |
| EP4 | 观察城门、混入宁州、试探陈老板 | 增加黄简偷衣后留补偿伏笔 |
| EP5 | 风水救人、身份到手、回北魏 | 增加陈老板权衡风险，以及身份文书办理过程 |

## 后续检查点

写 EP1 剧本前，需再次确认：

1. 乱葬岗冷开场是否保持“少解释，多悬念”。
2. EP1 结尾是否足够推动 EP2。
3. 老者临终信息是否过多。
4. 黄简的第一句关键台词是否符合克制人设。
5. 画面是否适合短剧/漫剧开篇视觉冲击。

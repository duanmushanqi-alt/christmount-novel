# 主角角色档案：黄简/黄繁/梁执

## 元数据
- **真实姓名**：黄简
- **化名1**：黄繁（陈汉国宁州城人士）

### 2.2 性格关键词

**核心特质**：

**生存技能**：

- 伪装潜入：改变身份、混入人群
- 观察分析：快速判断环境和人心

**智谋策略**：
- 障眼法：用帽子引开狱卒

- 应变能力：面对突发情况冷静应对

**武力能力**：

- 当前状态：十年牢狱导致身体虚弱，无法发挥
- 未来潜力：身体恢复后可能有简单动作戏

### 3.2 能力限制

**身体限制**：
- 十年牢狱导致体力虚弱
- 需要时间恢复体能
- （此时）不擅长武力对抗
- 有武学基础但无法发挥

**知识限制**：
- 道法知识仅为皮毛

**资源限制**：
- 初期身无分文

**次要欲望**：
- 重建家族荣耀
- 保护剩余的亲友
- 找回失去的十年

### 4.2 核心恐惧

**最深恐惧**：
- **无力感**：最怕再次陷入无能为力的境地
- **失去**：害怕再失去重要的人

## 七、行为准则

### 7.1 做事原则

**核心原则**：
1. **谨慎第一**：宁可慢一步，不可错一步
2. **观察先行**：了解情况再行动
3. **多重准备**：总有备用方案
4. **善良底线**：不伤害无辜
5. **果断赌博**：评估风险后敢于下注

**决策模式**：
- 收集信息 → 分析利弊 → 评估成功率 → 制定计划 → 准备后路 → **果断下注**

**底层智慧**：
- 作为相对底层的百姓，只能用这种方式复仇
- 在远处更大的敌人发现前，必须先干掉眼前的敌人
- 保证一定成功率后就会赌，不能等到万无一失

### 7.2 禁忌红线

**绝不触碰**：
- 不伤害无辜百姓
- 不轻易暴露身份
- 不冲动行事（但评估后会果断）
- 不放弃复仇

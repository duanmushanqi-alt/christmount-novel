# 角色语言档案

> 基于第1-15章实际写作内容提炼，记录主要角色的说话模式和语言特征

---

## 黄简（主角）

### 基本语言特征
- **语调**：低沉克制，话少但有分量
- **语速**：慢，每句话都经过思考
- **语言水平**：受过商家子弟教育，能文能武，但习惯用简短白话
- **十年狱中影响**：话更少了，废话近乎为零；沉默是常态，开口必有目的

### 说话习惯

**口头禅/高频表达**：
- "……"（沉默是最常见的"回答"）
- "继续说。"（追问，简短命令式）
- "嗯。"（确认，一个字）
- "我知道了。"（接收信息，不多评论）
- "先说……"（控制对话节奏时）

**称呼习惯**：
- 对林玄影：直接叫名字，或"你小子"（回到熟人模式后）
- 对陈老板等陌生人：不主动叫称呼
- 自称：我
- 伪装身份时：配合身份调整（黄繁身份时卑微农民腔调）

### 语言模式

**陈述**（简洁，只说结论）：
- ✅ "我没吓唬。"
- ✅ "那就算了。"
- ✅ "不要让任何人知道我还活着。任何人。"

**疑问**（追问式，不绕弯子）：
- ✅ "为什么是两年前。"（不带问号，更像命令）
- ✅ "然后呢？"
- ✅ "到底发生了什么事。"

**感叹/愤怒**（极少，但一旦出现力度极强）：
- ✅ "没想到什么？没想到我全家会死？没想到会害我一辈子。"
- ✅ "为什么？为什么要害我？"

### 情绪表达方式

- **高兴/放松**：几乎不出现在第一卷，最多是沉默中带一丝松弛
- **愤怒**：先是沉默，然后是身体动作（拳头），话反而更少
- **悲伤**：喉咙发紧，说不出话；或者只是"……"然后转移话题
- **思考时**：沉默，或者重复对方话里的关键词："大虫咬死了……"
- **紧张/危险时**：反而话多起来，但都是刻意为之（伪装卖菜农民时）

### 阶段性变化

| 阶段 | 章节 | 语言状态 |
|------|------|---------|
| 狱中 | 第1章 | 极简，每句话都有目的（吓唬狱卒时刻意营造诡异感） |
| 逃亡/伪装 | 第2-5章 | 伪装时能切换到卑微农民腔；内心独白比对话丰富 |
| 回兴城/调查 | 第6章 | 对街坊问话简短，不多透露 |
| 与林玄影重逢 | 第7-12章 | 对林玄影追问式，一句接一句，不给喘息 |
| 复仇确立后 | 第15章 | 命令式分工，身份宣言（"黄简已经死了"） |

### 标志性台词（第一卷）

```
"别动他。"（第1章，护住老者）
"守灵。给他守一夜，送他走。"（第1章，抓住机会）
"你们动吧。不过我提醒一句——他刚咽气，怨气正重。"（第1章，将计就计）
"对不起了……但我想活下去。"（第1章，独白）
"继续说，当年到底发生了什么。"（第11章，控制对话）
"不要以为死就能解脱了，你不帮我找出当年的真相，我做鬼都不会放过你的！"（第11章）
"从今往后——黄家的黄简，已经死了。死在十年前那条山道上。"（第15章）
```

## 林玄影（矛盾盟友）

### 基本语言特征
- **语调**：低沉沙哑（酒精长期损耗），不笑时带着阴郁
- **语速**：时快时慢，回忆当年时会停顿、磕绊
- **语言水平**：同样是商家子弟出身，但十年愧疚让他说话时常自我中断
- **心理状态的外显**：说话时会绕圈子、自我否定、突然停住

### 说话习惯

**口头禅/高频表达**：
- "对不起……"（反复出现）
- "我……"（起句常带停顿）
- "我当时不知道……"（逃避式辩解，随后往往自我打断）
- "……算了。"（自我否定）

**特殊语言模式**——对"幻觉"说话时（第7-8章）：
- 心防全开，反而更流畅
- "又来了，你这次换了个样子……陈汉国人？"（疲惫的接受，不再惊慌）
- 独白时语无伦次："都是我的错……害了你……害了所有人……"

### 情绪表达

- **愧疚时**：话多且碎，反复道歉，语句不连贯
- **被追问时**：先停顿，再硬撑着说，常常说到一半停住
- **被揍时**：不说话，或者只是"……我当时……"
- **唯一冷静的时刻**：汇报调查情况（第15章），反而清晰有条理——这是他两年来唯一能做的事

### 标志性台词（第一卷）

```
"又是幻觉……我怕是魔怔了。"（第7章，独自喃喃）
"都是我的错……害了你……害了所有人……"（第7章，独白）
"这次倒是换了个样子……陈汉国人？"（第8章，以为是幻觉）
"我……没想到会变成那样……"（第9章，被揍后）
"对不起，我当时不知道，我已经离开好几年了……"（第10章）
"他……没法不同意，那是他也死了……"（第10章，说林父死亡）
"我一定会的。"（第11章，承诺查明真相，这是第一卷里他唯一斩钉截铁的话）
```

## 狱卒（群像）

### 类型特征
两类狱卒形成对比：

**年轻狱卒**：
- 不耐烦，嘴碎，爱抱怨
- "真他娘的晦气。"
- "你懂什么……"
- 信鬼神但嘴上不承认，实际上比老狱卒更怕

**老狱卒**：
- 老油条，话少，但说的都是定论
- "早点弄完早点回去睡觉。"
- "闭上你的乌鸦嘴。"
- 见过世面，但骨子里更迷信

### 对话特点
- 两人说话经常是一长一短的配合
- 关键时刻老狱卒先表态，年轻狱卒附和
- 碰到鬼神话题，声音会不自觉压低或提高

## 陈老板（第4-5章）

### 语言特征
- **风格**：老练商人，客气但留三分距离
- **表面**：礼貌，"客官说笑了"，"老板我看您也是读书人"
- **本质**：每句话都在观察、试探，话里有刀

### 典型对话模式
```
"客官说笑了。"（拒绝但不得罪）
"老板，我看您也是读书人……"（在黄简展示知识后，态度转变的信号）
```

## 对话禁忌

### 黄简不会说的话
- ❌ 长篇大论的感慨和抒情（十年牢中磨掉了这个习惯）
- ❌ "我好累啊"、"我好痛苦"这类直接表达情绪的句子
- ❌ 主动解释自己的想法（他只说结论，不说过程）
- ❌ 绕弯子（话少但直接）

### 林玄影不会说的话（第一卷阶段）
- ❌ 轻松的语气（即便和黄简建立合作后，他依然阴郁）
- ❌ 过于清醒流畅地叙述当年（那是他会停顿、磕绊的地方）
- ❌ 为自己辩护（他不会这么做，只会自责）

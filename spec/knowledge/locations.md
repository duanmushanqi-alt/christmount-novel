# 场景地点（空间布置）

> **用途**：常出现场景的位置、格局、陈设预设。写新章节时引用本文件，避免同一场景在不同章节描写矛盾（如门朝哪开、桌子摆在哪、从书房到后门要经过什么）。
>
> **回填规则**：
> - 第一次写某场景时，按下方模板把布局**简洁**写下；不必面面俱到，只记会影响后续描写的关键项。
> - 后续章节修改了布局（如黄繁加了第二张桌子），**同步更新本文**，并在该条目注明"第X章变更"。
> - 与正文冲突时**以正文为准**，回头修订本文。

---

## 模板（每个场景 5 项即可）

```
### [场景名]

- **位置**：城内方位、相对哪条街/哪个坊
- **格局**：进出口、房间划分、与相邻空间的关系
- **陈设**：会被反复描写到的家具、摆件、门窗、光线来源
- **用途**：谁在此做什么（住、谈事、藏物等）
- **首次出现 / 关键变更**：第X章首现；第Y章新增/改动……
```

## 一、核心场景（已稳定使用）

### 承平巷小院（黄繁住所）

- **位置**：承平巷，巷子深处，独门独院。离集市走路约一刻钟。
- **格局**：前门临巷、后门通后巷；院内一进，附带一个杂物间。四周矮墙，邻居隔着一条窄巷。后门是黄繁夜出夜归的主要通道（绕路至林宅后巷）。
- **陈设**：屋内桌一张、油灯一盏（灯芯可拨小，光只够照亮桌面那一块）；窗有窗板可合严；墙角暗处藏石匣（重要物证存放点）。院子无显眼陈设。
- **用途**：黄繁租住的住所，签了半年租约。白天空着或周母代看铺子时偶尔回来；夜里子时从后门出入会林玄影。回家点灯前的标准动作：闩院门、合窗板、再点油灯。
- **首次出现 / 关键变更**：第26章首现（黄繁刚回兴城租下）。

### 黄繁铺子（干货店）

- **位置**：与承平巷小院分开的另一处铺面，房东介绍盘下来的（房东"隔壁街朋友"出手）。具体街名暂未明写——下次进入此场景前请确认或补一处。
- **格局**：临街小铺，铺面不大。柜台朝门口（黄繁可坐柜台后看进出的人）。前店一间，无后舍记录；周母有时来代看店。
- **陈设**：柜台、货架（堆山货：木耳、干笋、蘑菇等）。柜台是日常落点（看书、放东西、周顺把南瓜子包放在柜台上）。
- **用途**：黄繁的明面身份场所——干货商人。白天经营，客人不多时黄繁练观人术（在柜台后把进出的人一个个过一遍）。借口"出门进货"时实则研究堪舆/风水/符箓。周顺隔几天来此传话。
- **首次出现 / 关键变更**：第26章房东介绍提及；第27章黄繁去看；第30章已正式经营，收第一批山货尾款。

### 林宅（含后巷、院子、书房）

- **位置**：城内某处，黄繁从承平巷绕远路（"从东边转过去"）才到。林宅后面是一条窄巷，两边各家后墙。
- **格局**：
  - **后巷**：很窄，两边各家后墙，偶有树枝伸出。林宅这段墙角有老槐树枝丫伸过墙头，可作辨识标志。
  - **翻墙点**：老槐树旁边的墙角，墙上有几个踩出来的浅坑（黄繁第26章首次翻入留下，可重复用）。
  - **后门**：可叩门进入。暗号：叩三下停一息再叩两下（第29章后稳定为此暗号）。门有门闩。
  - **后院 → 书房**：穿过后院、廊下走到书房。院内有小天井，铺青石板，几盆半死不活的花草。
  - **书房**：在西边。窗纸糊，灯光会映出人影（外面看得见）。门可闩。
- **陈设**：书房内有桌、灯、茶（林玄影常先沏好茶等）。
- **用途**：林玄影的住处与情报中枢；与黄繁子时密会的地点；情报汇总、下一步部署的讨论地。
- **首次出现 / 关键变更**：第26章首现（黄繁夜里翻墙入院找林玄影）；第29章起改为后门叩门暗号入；约定碰面方式：书肆传纸条 → 后门叩门。
- **历史/相关**：林玄影另有一处铺子，第26章前后被烧，正在重修——非住所，是另一处财产/掩护点。

### 周家租屋

- **位置**：进城后租住，具体街名暂未明写。第30章林玄影帮"找更近的租屋"，相对铺子的位置可写得更紧凑。
- **格局**：（待回填——具体几间屋、是否带院、采光通风等首次详写时补）
- **陈设**：周母做缝补浆洗的活计场地；妹妹养病的位置（第30章"妹妹聪慧"的细节场景化展现的物理空间）。
- **用途**：周顺一家居住。母亲接缝补浆洗活、妹妹养病、黄繁送药/送钱时偶尔进入。
- **首次出现 / 关键变更**：第27章口述提及（搬到城里小半个多月）；第30章林帮换更近的租屋——**这里有过一次搬迁**，回填时分清新旧地址。

### 赵掌柜杂货铺（周顺早期工作处）

- **位置**：周顺干活的那条街上（第28章："南边、西边大多数街都天天走的，我干活的铺子就在昨天那条街上"）。从铺子出来 → 往南走到街口 → 右拐 → 进集市东门，沿途有一根可作标记的柱子（第28章周顺画记号处）。
- **格局**：（无详细描写需求，是临街铺面，周顺搬货跑腿即可）
- **陈设**：（无）
- **用途**：周顺早期生计（一个月几百文）。第30章起周顺已不再做此活，偶尔路过帮忙。
- **首次出现 / 关键变更**：第27章首现；第30章周顺已离职。

### 旧书肆（情报传递点）

- **位置**：城内书肆，具体未明写。
- **格局**：进门左边新书架，右边两排旧书架靠着墙。掌柜坐柜台后拨算盘，伙计捆书。
- **陈设**：旧书架底层那排积灰，平时没人翻——是固定的"死信箱"位置（靠墙最里数第三本是约定本）。
- **用途**：黄繁与林玄影之间的暗号传递点。约定本里夹纸条传递碰面时间（如"明晚亥时，后门，叩三下停一息再叩两下"）。
- **首次出现 / 关键变更**：第29章首现并确立为情报点。

## 四、使用规则

1. **写新章节前**：检查本章会进入哪些场景，对照本文件确认布局——若发现要写的内容与本文冲突，先停下来决定改哪个。
2. **写新章节后**：若新增了空间细节（黄繁在铺子角落加了书架），立刻同步本文。
3. **场景退场不删除**：保留历史条目（如监狱），便于后续回忆/对照写作时调用。
4. **本文件不写氛围/感官比喻**：那些是写作时的发挥，本文只记**空间事实**。氛围素材见 `style-reference.md`。

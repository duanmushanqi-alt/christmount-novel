# 术语速查卡

## ⚡ 最常用术语

| 类别 | 错误 ❌ | 正确 ✅ |
|------|---------|---------|
| **地名** | X城 / X地 | 兴城 |
| **宅邸** | 黄府 / 林府 / 陆府 | 黄宅 / 林宅 / 陆宅 |
| **主角名** | 第1-47章（第一、二、三卷） | 黄繁（陈汉国身份） |
| | 第48章起（第四卷） | 梁执（C国贵族） |
| **国家** | A国 / B国 / C国 | 北魏 / 陈汉国 / 西蜀 |

## 🎭 四仇敌简称

- 陈伯庸 → 陈刺史（地方层）
- 陆识 → 陆家养子（商界层）
- 顾从戎 → 兵部侍郎（朝堂层）
- 萧策 → 三皇子（最终BOSS）

## 📍 常用地点

- 兴城：兴州首府，主角故乡（北魏）
- 兴州：陈伯庸辖区（含兴城、宣城、乐城等）
- 宣城：兴州下辖，K监狱所在地
- 乐城：兴州下辖，陈新豪早期销金窟
- 邳州沄城：邻州，林家衰落后投奔处
- 宁州城：陈汉国城市，黄简获得新身份处

## ⏰ 时间表达

- 十年前：黄简19岁，入狱前
- 狱中十年：19岁→29岁
- 十年后：黄简29岁，出狱
- 两年前：林玄影回兴城（第27年）
- 去年：街坊说林玄影回来的时间
- 五年前：黄父自杀（第25年）

## 🏠 建筑类型

- 商人：宅（黄宅、林宅、陆宅）
- 官员：府（县令府、刺史府）
- 皇室：宫

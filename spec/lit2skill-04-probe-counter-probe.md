# Skill 4：试探与反试探

> 本作系列 lit2skill 改造版第 4 号技法。主要借鉴来源：《琅琊榜》。

## description（A2 未来触发场景）

主角与陌生人首次社交接触 / 与潜在怀疑者对话 / 情报搜集但不能露底 / 被怀疑时的应对 / 伪装身份的对外交流。

## I 骨架

每次社交接触不是聊天而是**攻防**。双方都在试探对方的底（身份/意图）、隐藏自己的底。高阶技法：把对方的试探**反向利用**——顺着他的试探**种一个假印象**进去。

## A1 可对照的场景

- 琅琊榜：誉王 × 梅长苏朝堂试探
- 本作第 16 章茶肆：赶车人 × 黄繁的信息盘问，主角的"行脚的"含混回答 + 事后独坐一里复盘
- 本作第 30 章：林玄影 × 镖头酒桌，信息获取包裹在生意闲聊中

## E 可执行步骤

- **E0 前置判断**：双方任一方有**隐藏动机**？（纯闲聊不用本 skill）
- **E1 四份清单**：
  - A 想知道什么 / A 想隐藏什么
  - B 想知道什么 / B 想隐藏什么
- **E2 问题链**：每个问题表面像闲聊，实际在探某一点；每个回答表面像顺口，实际在堵漏
- **E3 反制时刻**（可选但推荐）：被试探的一方抓住试探的钩子，**反向种信息**——让对方以为自己套出了真相
- **E4 复盘**：场景结尾或稍后独处时，主角内化复盘——哪里露了？守住了什么？种下了什么？下次改什么？

## 诊断钩子

1. **删掉这场对话**故事是否完全成立？（完全成立 = 不是攻防，是填充）
2. 四份清单的每一项都在对话里被执行了吗？
3. 有没有反制时刻？没有 = 单方攻击，张力层次薄
4. 事后有没有复盘？没有 = 读者不知道谁赢了谁输了
5. 复盘时主角是否承认破绽？（全赢 = P-03 剧情倒推嫌疑）

## B 边界与盲点

- 真朋友之间不用（本作黄繁 × 林玄影、黄繁 × 周顺之间不适用）
- 过度使用 → 每段对话都紧张 → 读者疲劳。**商业日常对话**大多不用本 skill
- 反制时刻用多 → 主角显得无敌 → 必须允许偶尔被识破或失手（第 16 章茶肆就是典型失手示例）
- 每场都复盘 → 拖沓；有些攻防让读者**自己从动作里读出**谁占了上风

## 正反对照

- 平庸：赶车人问"从哪来"→ 主角"陈汉国"→"哦"→ 场景结束（信息交换完就散）
- 本作第 16 章：每一问背后是试探（"一个人走？""往哪去？""西边哪儿？"），每一答卡在守底/露底之间；走后独坐复盘"'行脚的'一出来就不对"——**失手版攻防**

## 与 pitfalls 交叉索引

- E1 四份清单没列 → 常表现为 P-22（情绪内容时机错位，本该紧绷的对话写成了散漫闲聊）
- E3 反制过度 → P-03（剧情倒推角色，让对手突然变蠢配合主角反制）
- E4 复盘缺 → 与 P-01 互为背景，信息边界失守会让主角"莫名其妙就赢了"

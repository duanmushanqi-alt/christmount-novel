# Skill 5：敌人视角合理性

> 本作系列 lit2skill 改造版第 5 号技法。主要借鉴来源：《琅琊榜》。

## description（A2 未来触发场景）

写敌人/对手反应场景 / 写主角布局需要敌人按预期反应 / 写敌人角度独立段落 / 写复杂配角的选择 / 侧写敌人的政治/经济动作。

## I 骨架

敌人不是为主角剧情服务的工具。他们有自己的**情报体系、判断逻辑、利益结构**。主角的聪明是相对的——敌人全是降智 NPC，主角的谋划就廉价。敌人在他自己的信息域内**完全合理**，主角胜利才有分量。

**硬指标（V3 独特性所在）**：这个敌人的决策在他信息域内完全合理。不是"他也挺聪明"，是"把主角拿掉后他的行为仍然独立成立"。

## A1 可对照的场景

- 琅琊榜：谢玉、夏江在各自信息域内运作

## E 可执行步骤

- **E0 前置判断**：这个敌人/配角在这场戏里的**信息域**是什么？（知道什么 / 不知道什么 / 误判什么）
- **E1 列三基本面**：
  - **利益**：想要什么？怕失去什么？
  - **信息**：知道哪些事实？误判哪些？
  - **性格/能力**：聪明/草率；大胆/胆小；细致/粗疏
- **E2 最合理选择**：让他基于三基本面做出**对他而言最合理**的选择——哪怕这选择恰好帮了主角
- **E3 删主角检验**：把主角和主角线全部删掉，这敌人的行为是否**独立成立**？成立 = 通过；必须主角在场才解释得通 = 工具人，回炉

## 诊断钩子

1. 能说清这敌人"为什么这样做"吗？（动机链 + 信息链）
2. 他的选择是"他认为最好的"还是"作者需要他这样"？
3. 他会不会犯错？好的敌人会犯错，但错在**他能力/信息范围内的错**，不是被作者摁着做蠢事
4. 如果我是他，基于三基本面，我会这样做吗？
5. 删主角检验能过吗？

## B 边界与盲点

- 次要 NPC 不需完整三基本面（卖烧饼大婶只需一个动机：做生意）
- 短戏份配角可简化为单一动机（好奇、自保、贪便宜）
- 过度合理 → 敌人太强主角赢不了；适度允许**信息盲点 + 情绪偏差**（愤怒、傲慢、轻视导致的误判）
- 本作典型：陈伯庸不在意"愚民声望"是他的真实判断，但这个判断让他低估民间裂缝——**信息域内合理的盲点**

## 正反对照

- 平庸：老狱卒遇到黄简 → 剧情需要信息 → 他就说了
- 本作：老狱卒的**自保逻辑**——防惹事、防说错话被追究 → 不想多说 → 但不敢得罪 → 说一半留一半
- 平庸：敌人因"剧情需要"去某地
- 本作第 30 章：陈伯庸把精力放在祥瑞后续、不亲自出面处理邳州家属 = 基于他的**利益排序**与**经验**（他以为愚民好糊弄）

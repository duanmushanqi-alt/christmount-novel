# Skill 2：代理行动

> 本作系列 lit2skill 改造版第 2 号技法。主要借鉴来源：《琅琊榜》。

## description（A2 未来触发场景）

主角需要同时在多线出场 / 亲自出手会暴露身份 / 故事要"权谋感"而非"武力感" / 想让配角有独立存在感和分工。

## I 骨架

核心角色不亲自执行关键动作，通过**代理人**完成。主角的"在场"由**决策—部署—复盘**三段体现，不由动手体现。代理人的**能力局限和判断 variance** 本身是戏剧燃料。

## R 原文参考（待二轮取证）

转述自 style-reference.md："梅长苏很少亲自出手，通过蒙挚、飞流等人执行"。**V1 未完成书内多处独立取证**。

## A1 可对照的场景

- 琅琊榜：飞流动手 / 蒙挚传信 / 甄平处理外围，梅长苏留在谋局位置
- 本作第 30 章：黄繁通过周顺（情报网）、林玄影（地面执行）、镖局、抄书吏分线收集，自己在铺子里用观人术

## E 可执行步骤

- **E0 前置判断**（v1.1 新增，防误触发）：识别本场戏的**核心功能**
  - 权谋 / 信息战 / 布局执行 → ✅ 调用本 skill
  - **人设建立**（展示主角独有能力、第一次显手段）→ ❌ **不调用**
  - **关键情感戏**（主角情感出口、亲密关系锚点）→ ❌ **不调用**
  - 世界观 / 日常 → 视情况，通常不强用
- **E1 分派代替执行**：主角做决策后**不直接写执行**，写"分派"——交代谁做、为什么由他做
- **E2 切代理人视角**：执行场景切到代理人视角或第三方旁观视角，主角退到幕后
- **E3 保留执行 variance**：代理人按自己的理解和能力做，允许偏差（做过头 / 做不到位 / 做出自选动作）
- **E4 主角复盘**：事后由主角复盘——此时读者才知道"哪里合预期、哪里走偏"

## 诊断钩子

1. 先过 E0——本场戏核心功能属于哪一类？不在适用区就停用
2. 主角在这场里亲自做了什么关键动作？能换成代理人做吗？
3. 代理人有**自己的判断**吗？（没有 = 不是代理人，是提线木偶）
4. 执行有 variance 吗？（次次完美 = 戏剧张力为零）

## B 边界与盲点

- 主角前期无班底时用不上——要先攒人（本作第 1 卷、第 2 卷前半段不用）
- 打怪升级类武侠用不上（动手本身是卖点）
- 过度使用 → 主角"悬浮"——在**情感戏 / 关键转折 / 第一次杀人**这类节点必须让他亲自下场
- 代理人全是工具人 → 失去独立人格（参考 pitfalls P-17 互动缺情绪弧线、P-03 角色不能被剧情倒推）

## 正反对照

- 平庸：主角亲自潜入、亲自审问、亲自动手——高风险无层次
- 本作第 13 章岔道救母女：**核心功能是人设建立**（首次展现牢里学到的"看人"本事、"三分猜七分唬"的方法论）→ E0 判定**不调用本 skill**，必须主角亲自
- 本作第 30 章布局：**核心功能是权谋铺网** → E0 通过 → 通过周顺/林玄影/镖局/抄书吏多线代理执行，主角坐镇分析

## 与 pitfalls 交叉索引

- E0 误触发 → 常表现为 P-03（剧情倒推角色，为了"让主角不出面"牺牲人设戏的张力）
- E3 variance 为零 → 接近 P-21（NPC 指路感，代理人变成"任务 NPC"）
- E2 视角切换需注意 P-01（全知视角污染），代理人视角里不能泄露主角未告知的信息

# Skill 3：情感延迟与余震

> 本作系列 lit2skill 改造版第 3 号技法。主要借鉴来源：《琅琊榜》。v1.1 扩容版。

## description（A2 未来触发场景）

主角重大创伤情绪 / 久别重逢 / 死别告别背叛 / **重大紧张与恐惧**（如首次伪装失败的复盘）/ **愧疚与沉压情绪** / 任何情感浓度爆表的场景。

**注**：v1.1 从"只用于悲伤类"扩容到所有高压情绪。

## I 骨架

强烈情感**不在触发点当场释放**，而在触发后的**侧面时刻**（独处、做别的事、一个无关小物件）以**小动作 / 生理反应**溢出。主要释放后还有**一次余震**——在后续场景里被一个小触发带出，不再言明。

原理：当场爆发 = 情绪廉价；延迟释放 = 压力积累 + 读者参与（读者脑补没写的那层）；余震 = 把情感从"事件"做成"状态"，让角色带着情绪活下去。

## A1 可对照的场景

- 琅琊榜：梅长苏与靖王重逢后的大量留白
- 本作第 13 章祭拜全章：桂花糕触发、回忆穿插、抹泪无声、回城看母女背影的二次涌起
- 本作第 16 章：茶肆伪装失败后，走出一里地在路边石头上坐下——紧张过后的复盘与发抖

## E 可执行步骤

- **E0 前置判断**：本场是否属于高压情绪爆发点？触发点是否值得做延迟处理？（日常小情绪直接写，不用本 skill）
- **E1 触发点冷处理**：写动作、写环境、写物件，不写心理抒情
- **E2 侧面释放**（v1.1 扩容）：
  - 可以切到**下一场**（物件 / 气味 / 闲话触发）
  - 也可以在**同场内切入回忆或新物件**作为侧面通道（第 13 章祭拜即属此类）
- **E3 只用 1-2 个小动作**：手指收紧、停顿、把碗放下、一滴接一滴的流
- **E4 先控制再失控**：主角当场**第一反应仍要显露**，只是不彻底释放
- **E5 不解释——三色分级**（v1.1）：
  - 🔴 禁用：抽象情绪词（悲痛、思念、愤怒、委屈、心如刀绞、悲从中来）
  - 🟡 节制：**对比式注解**（例如"昨晚他撑住了……可到了这里什么都撑不住了"）——不描述情感本身，只做"克制对比"，可用但每场不超 1 处
  - 🟢 推荐：纯动作 / 环境 / 生理反应
- **E6 情感余震**（v1.1 新增）：主要释放之后的 1-2 个后续场景内，安排**一次小触发**（相似的人 / 相似动作 / 相似气味）让情绪**被带出而非言明**。余震不再是爆发，是回声

## 诊断钩子

1. 触发点写了"红了眼眶 / 泪水夺眶而出 / 心如刀绞"？→ 🔴 删掉，搬到侧面时刻
2. 溢出段出现抽象情绪词？→ 🔴 换成动作
3. 读者需要你**告诉**他"这是什么情感"？→ 前面铺垫不够，不是后面写得不够
4. 主要释放后 1-2 场戏内有没有"一秒回声"？（没有 = 漏 E6）
5. 🟡 对比式注解超过 1 处？→ 压缩

## B 边界与盲点

- 不是所有情感都要延迟——日常小情绪直接写
- 间隔超过 2-3 个场景读者会忘——余震必须及时
- 滥用 → "装深沉"；偶尔**该当场爆发时就要当场爆发**（例如第 7 卷末主角第一次杀人后的情绪）
- 与 E6 的冲突：如果故事节奏要求下一场紧接高密度新信息，可主动放弃余震，不硬安

## 正反对照

- 平庸："他跪在坟前，眼泪如断了线的珠子，十年的委屈全在这一刻涌上心头"

## 与 pitfalls 交叉索引

- E5 🔴 ≈ P-24（情绪质感不区分，所有情绪都用"什么东西往上涌"）
- E4 先控制再失控 ≈ P-29（崩塌情绪的多种形态），周顺式"强撑把事做完"也属此类
- E6 在梗概里的指示 ≈ P-31（梗概情绪词使用标准），描写情绪的功能性词语可用

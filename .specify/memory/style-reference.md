# 风格参考

> 本作风格DNA + 参考作品借鉴指南。与 spec/writing-style-guide.md 配合使用：本文件定方向和参考坐标，那边定具体技法和检查标准。

---

## 一、本作风格DNA

### 1.1 叙事质感：冷硬写实
- **白描为主**：场景靠具体细节（物件、气味、温度、声响）立起来，不靠形容词堆砌
- **短句主导**：节奏干净利落，长句只用于铺垫和过渡
- **生理反应代替情绪标签**：不写"他很紧张"，写"心跳一下比一下快""手心出汗""后背发僵"
- **动作暴露心理**：不写"他心虚了"，写"把碗放下，在腿上蹭了蹭"
- **粗粝感**：人物会说废话、做多余动作、有杂念——不要写得太"干净"

### 1.2 对话：吝啬到位
- 每句对话必须有信息量或推进关系
- 身份区分：农民用"俺""老爷"，商人客气精明，官差简短居高临下
- 标签用"说""问"为主，动作替代比"XX道""XX叹道"更自然
- 问答节奏紧凑，一问一答推进，不拖

### 1.3 环境描写：功能性优先
- 每处描写必须服务叙事（推进情节/暗示信息/营造氛围）
- 季节天气时辰是刚性约束
- 多感官调动（视觉+听觉+触觉+嗅觉），不只是视觉
- 点到即止，一两句定调，不铺陈

### 1.4 节奏模式
- **紧张段落**：短句堆叠，动作密集，不插内心独白
- **思考段落**：放慢，允许内心推理，但碎片化而非长篇大论
- **过渡段落**：用具体行为（吃饭、赶路、整理物品）自然过渡
- **高潮后留白**：情绪爆发后必须有沉默/脱力段落

## 二、参考作品分析

### 2.1 《琅琊榜》（海宴）—— 取其骨

琅琊榜的核心技术是**信息差的精密管理**：读者知道梅长苏的真实身份，朝中绝大多数人不知道。每一步棋，读者能看到表面动作和底层意图的双重含义。这种"观众比角色多知道一层"的结构制造持续张力。

**可借鉴的技法：**

| 技法 | 琅琊榜实现方式 | 本作应用 |
|------|--------------|---------|
| 信息差叙事 | 读者知道梅长苏=林殊，看他与旧人互动时有双重张力 | 读者知道黄繁=黄简，看他与老狱卒、路人互动时利用同样张力 |
| 代理行动 | 梅长苏很少亲自出手，通过蒙挚、飞流等人执行 | 黄简通过林、小偷等人在兴城行动，自己在另一条线上 |
| 试探与反试探 | 誉王试探梅长苏底细，梅长苏反过来利用试探传递假信息 | 茶肆对话、老狱卒偶遇——每次社交接触都是攻防 |
| 敌人视角合理性 | 谢玉、夏江有自己的情报体系和判断逻辑 | 陈伯庸对林调查的反应（轻视但合理）、老狱卒怕事的自保逻辑 |
| 克制的情感表达 | 梅长苏与靖王重逢后的大量留白 | 黄简祭拜父母（第13章）的克制写法，延迟的情绪释放 |
| 方寸之间见格局 | 一次朝会、一封密信就能让权力格局偏移 | 一次茶肆闲聊就能暴露身份破绽，一个村庄的异常就能牵出十年旧案 |

**不照搬的部分：**
- 琅琊榜文笔偏雅，带古典韵味——本作更糙、更接地气
- 梅长苏从一开始就有资源和班底——黄简白手起家，前期更狼狈
- 琅琊榜以庙堂博弈为主——本作前期是底层/江湖视角，到后卷才进入庙堂

### 2.2 《庆余年》（猫腻）—— 取其肉

庆余年的范闲之所以讨喜，不是因为他多强，而是因为他**真实**：有小聪明也会犯错，对身边人有真感情，面对强敌先怂再想办法。猫腻的另一个强项是**世界观通过生活本身传达**，从不灌输设定。

**可借鉴的技法：**

| 技法 | 庆余年实现方式 | 本作应用 |
|------|--------------|---------|
| 生存智慧的笨拙展示 | 范闲在各方势力间周旋，靠察言观色+快速适应 | 黄繁在路上学习伪装：茶肆被问后复盘、编身份故事、一次次练习——笨拙但认真 |
| 收服人心的真实感 | 范闲对五竹、滕梓荆不是"收下属"而是"交朋友" | 黄简帮小偷一家不是施舍，是"看到值得投资的人"——有功利心但底色善良 |
| 危险中的自嘲 | 范闲紧张时偶尔冒出不合时宜的念头 | 黄繁独自赶路时的自我复盘（"行脚的——这词一出来就不对"） |
| 世界观融入日常 | 通过吃穿住行让读者自然理解设定 | 第16章买路程图、烧饼、茶肆、山神庙——不解释古代设定，就展示古代人怎么过日子 |
| 配角的独立人格 | 每个配角都有自己的小心思和行为逻辑 | 赶车人的好奇是闲人正常反应，不是"推动剧情的NPC" |
| 细节的温度 | 滕梓荆妻儿、侯季常的家书——让"人"从"棋子"中跳出来 | 小偷一家的困境不是苦情工具，是有温度的底层生活切片 |

**不照搬的部分：**
- 庆余年有穿越者视角的现代感和吐槽——本作没有这层，要更沉
- 庆余年后期有大量武力场面的"爽"——本作的"爽"来自智谋布局
- 猫腻文笔偏文艺、有诗意——本作更粗粝、更直，不追求漂亮句子

### 2.3 综合定位：取琅琊榜的骨，取庆余年的肉

- **骨**（结构层）：从琅琊榜学习信息差管理、布局层次、代理行动、敌人视角的合理性
- **肉**（质感层）：从庆余年学习让主角真实而非完美、让配角有自己的温度、让世界观通过生活传达
- **皮**（本作独有）：比两者都更粗粝、更底层、更冷硬。这是一个刚从牢里出来的人的世界——没资源、没背景、连说话都会露馅。那种"从零开始"的生存质感，是本作区别于两部参考作品的独特之处

## 三、场景类型写法速查

### 3.1 赶路/独处
- 写身体感受（脚疼、饿、冷、累）和实际问题（干粮还剩多少、水壶快空了）
- 内心以复盘和计划为主，不抒情
- 环境点到即止
- 参考：第16章全章

### 3.2 社交试探
- 每次对话都是攻防：对方在试探什么？主角在隐瞒什么？
- 事后必须有复盘：哪里露了？下次怎么改？
- 紧张感靠身体反应传达
- 参考：第16章茶肆、琅琊榜朝堂试探

### 3.3 信息交汇
- 两人把各自拼图拼在一起
- 一条一条放信息，每放一条停一停让读者消化
- 角色反应分层：A的反应 → B看到A反应后的判断
- 参考：第10-12章黄简和林的长对话

### 3.4 高压/生死
- 外松内紧：外表冷静甚至缓慢，生理反应暗示极度紧张
- 短句、动作密集，不插内心独白
- 感官替代视觉（被埋/躲藏时用听觉、触觉、嗅觉）
- 参考：第1-3章逃狱系列

### 3.5 情感爆发
- 延迟释放：先写控制，再写失控
- 小动作代替大描写（手指收紧、声音变低、转身出去）
- 爆发后留白：沉默比语言有力
- 参考：第9章黄简打林、第13章祭拜

### 3.6 底层生活切片
- 不俯视，不煽情——平视地写困难
- 具体的窘迫比抽象的"穷"更有力（饼硬得费牙、被褥薄得不挡风）
- 人物在困境中的尊严感和韧劲，不用旁白解释
- 参考：第16章山神庙过夜、庆余年底层角色段落

## 四、反面教材速查

### 4.1 一眼AI的特征
- 情绪标签化："他感到愤怒/悲伤/震惊"
- 堆砌式修饰："温暖而坚定/冰冷又带着决然"
- 过度解释：对话后加大段心理分析
- 万能短语："某种说不清的感觉""莫名的安心"
- 升华强迫症：每个场景结尾都要"他忽然明白了什么"
- 对称句式："XX不是XX，而是XX"

### 4.2 手感偏差警告
- **太"好看"**：句子太整齐、太对仗、太有文学腔 → 弄粗糙一点
- **太"聪明"**：角色分析问题一针见血、从不犯错 → 让他犯蠢、反应慢
- **太"完整"**：每个场景都有起承转合 → 允许断裂和留白
- **太"干净"**：人物从不说废话、做多余动作 → 加入杂质和日常碎片

# 章节梗概写作规范

## 一、模板结构

每章梗概按以下顺序排列：

### 1. 头部信息

```
# 第XX章：章名 — 章节梗概

**视角**：XXX（单视角/多视角说明）
**时间**：与上章的时间关系
**地点**：主要场景
**预估全章字数**：XXXX-XXXX字
```

### 2. 前置状态

一段话交代：上章结束时各角色状态、本章起点条件。只写与本章直接相关的。

### 3. 各段（叙事草稿 + 本段功能块）

每段结构：
- **叙事草稿**：用接近正文的语感写出段落内容，要有具体细节
- **本段功能块**（以 `**本段功能**：` 开头）

本段功能块写清楚三件事：
1. **写什么**：这段承载的信息/情节功能
2. **怎么写**：具体写法要求
3. **不写什么**：明确的禁区

### 4. 质量自检（Q1-Q8）

- Q1 角色驱动：行为是否由角色处境和性格驱动
- Q2 信息边界：角色应知/不应知是否严格区分
- Q3 能力一致：能力水平与行为是否匹配
- Q4 反派智商：反派行为是否合理（若本章涉及）
- Q5 物理合理：世界观内的物理逻辑
- Q6 因果前置：伏笔和信息源是否可追溯
- Q7 信息不重复：是否避免了复述前章内容
- Q8 用语检查：是否符合反AI规范

### 5. 视角提醒

按角色列出：
- 每个出场角色：本章知道什么、不知道什么
- 读者：获得什么信息、什么悬念留存

### 6. 节奏设计表

| 段落 | 节奏 | 字数预估 |
|------|------|---------|

加一句整体节奏概括。

### 7. 已确认事项

编号列表，锁定本章所有关键决策。

## 二、方法论速查表

### 通用规则（所有章节）

| 规则 | 说明 |
|------|------|
| 动作替心理 | 内心状态只通过行为体现，不直说（"攥紧书角"而非"他很焦躁"） |
| 信息不点明 | 叙述事实和细节，不替读者解读价值 |
| 高点要轻 | 情绪最高点越轻越有力，不加渲染 |
| 时间标记用物象 | 用具体环境变化体现时间（"货换了两三茬"），不用抽象数字 |

### 动作/探索章

| 规则 | 说明 |
|------|------|
| 失败必写 | 能力展示前先写真实失败，失败有物理细节（闷响、卡住、歪了） |
| 触感声音分写 | 物理反馈逐个写，不笼统概括（三声各有质感差异） |
| 环境制约 | 黑暗/天气/地形增加操作的真实难度 |
| 章末极简 | 不回顾、不感慨、不复述，用一个细节收束 |

### 对话/信息整合章

| 规则 | 说明 |
|------|------|
| 推论vs跳跃 | 标注清楚哪些有据可循、哪些是角色主观投射 |
| 信息分层 | 区分"新增量"和"旧信息重新定性" |
| 话题转向 | 情感高点后两人不深说，转回正事 |
| 信息包装 | 信息获取包裹在角色正常行为中，不设专门调查环节 |

### 状态/过渡章

| 规则 | 说明 |
|------|------|
| 行为为主 | 状态交代用具体行为，不加分析句 |
| 多线快进 | 多条线用简洁段落各自推进，不拉平节奏 |
| 松弛与压力对比 | 日常细节和深层焦虑交替，不全松不全紧 |
| 悬念克制 | 提出问题但不给确定结论，让读者参与推断 |

### 地理/踏查章

| 规则 | 说明 |
|------|------|
| 伏笔轻种 | 路上见闻只是一句闲话带过，读者自己留印象 |
| 不适通过行为 | 身份/环境不适通过小动作体现（回头看、手松了松） |
| 因果真实 | 问路、选路线的逻辑要合理，不为剧情方便 |
| 单层叙述 | 旅途段不需要旁白解释，只写行动和环境 |
